Overview:
Campsite Commander is a mobile application designed to help campers organize and manage their camping gear efficiently. The application allows users to create, track, and review packing lists, ensuring that essential items are not forgotten before a trip.

Features:
View a dashboard showing the total number of packed items.

Add new camping gear with item details.

Categorize equipment for better organization.

Record item quantities and additional notes.

View a detailed checklist of all packed gear.

Validate required fields before saving items.

Application Screens:
Splash Screen:
Displays the application logo and loading screen before the user enters the application.

Main Dashboard:
Provides an overview of the packing list, including:

Total items packed

Quick access to add gear

Access to the detailed checklist

Preview of recently added items

Detailed Checklist Screen:
Displays all recorded camping gear with:

Item name

Category

Quantity

Additional notes

Purpose:
The purpose of Campsite Commander is to simplify camping trip preparation by providing an organized inventory system that helps users track and manage their camping equipment effectively.
